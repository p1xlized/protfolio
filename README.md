## My Megamind Genius Porfolio (˶ᵔ ᵕ ᵔ˶)

This is my over the top portfolio project, built with the latest and greatest in web development. It's a monorepo that includes a frontend, backend, and blog.

---
### 01. Monorepo_Structure
```
    /apps/web: The Frontend. Built with React, Vite, and TanStack Router.

    /apps/api: The Backend. Powered by Bun + ElysiaJS.
    
    /apps/blog: The Backend. Powered by Bun + ElysiaJS.

    
    // shared packages for both web and api


    /packages/config: Shared ESLint, TypeScript, and Tailwind configurations.
    
    /packages/db: Shared Drizzle config.
    
    /packages/types: Unified Type-safety across the entire stack (End-to-end Type-safety).
    
    /packages/ui: Shared Shadcn/ui components and Tailwind utilities.
```
